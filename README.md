# Dynamic-data-analytics-dashboard
🚀 Project Overview

The Dynamic Data Analytics Dashboard is an interactive web application built with Streamlit. It allows users to analyze and visualize datasets across multiple domains including:

E-commerce: Customer behavior, purchasing trends, product analytics
Student Performance: Grades, attendance, and progress metrics
Healthcare: Patient data, hospital statistics, and health trends
Sales: Revenue, product performance, and business KPIs

The dashboard automatically detects columns, generates interactive visualizations, and calculates key metrics to help users derive actionable insights in real-time.

✨ Key Features
✅ Multi-domain dataset support (E-commerce, Student, Healthcare, Sales)
✅ Dynamic charts and KPIs for quick insights
✅ Interactive filtering for targeted analysis
✅ Correlation heatmaps to detect relationships between variables
✅ Summary statistics for easy understanding of datasets
✅ User-friendly interface for non-technical users
🛠 Technology Stack
Python – Core programming language
Streamlit – Web interface and dashboard
Pandas & NumPy – Data manipulation and computations
Plotly / Matplotlib / Seaborn – Interactive and static visualizations

⚙️ Installation & Setup
Clone the repository:
git clone <your-repo-link>
cd analytics_dashboard
Install dependencies:
pip install -r requirements.txt
Run the application:
python -m streamlit run app.py
Open in browser:
http://localhost:8502/
