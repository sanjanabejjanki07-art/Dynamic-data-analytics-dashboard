# 📊 Dynamic Multi-Domain Data Analytics Dashboard

## 🚀 Project Overview

This project is a Dynamic Data Analytics Dashboard built using Streamlit.
It allows users to analyze datasets across multiple domains:

- E-commerce: Customer behavior, sales trends, product analytics
- Student Performance: Grades, attendance, and progress metrics
- Healthcare: Patient data, hospital statistics, and health trends
- Sales: Revenue, product performance, and KPIs

The dashboard automatically detects columns, generates interactive visualizations,
and calculates key metrics to provide actionable insights in real-time.

---

## ✨ Key Features

- Multi-domain dataset support (E-commerce, Student, Healthcare, Sales)
- Dynamic interactive charts and graphs
- Key Performance Indicators (KPIs) for instant insights
- Smart filtering options for targeted analysis
- Correlation heatmaps to detect relationships
- Summary statistics and automated column detection
- Optional predictive feature for forecasting trends

---

## 🛠 Technology Stack

- Python – Core programming language
- Streamlit – Web app interface and dashboard
- Pandas & NumPy – Data processing and computations
- Plotly / Matplotlib / Seaborn – Interactive visualizations
- Scikit-learn (optional) – For predictive modeling

## ⚙️ Installation & Setup

1. Clone the repository:
git clone <your-repo-link>
cd analytics_dashboard

2. Install dependencies:
pip install -r requirements.txt

3. Run the app:
python -m streamlit run app.py

4. Open in browser:
http://localhost:8502/
